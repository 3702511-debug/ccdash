# CC Dashboard

Локальный веб-обзор активных Claude Code сессий + удалённый доступ с iPhone/Android через PWA.

**Что умеет:**
- Видит все запущенные `claude` в терминалах Mac
- Шлёт сообщения, прерывает (Esc), голосовой ввод через whisper.cpp
- Drag-drop файлов, превью аттачментов
- Push-уведомления когда claude закончил и ждёт ответа
- Удалённый доступ с телефона (через твой VPS или Tailscale)
- Логин+пароль + биометрия через iOS/macOS Keychain

**Что нужно иметь:**
- Mac (M-серия или Intel)
- [Bun](https://bun.sh) (`brew install bun`)
- Опционально для удалённого доступа с телефона: VPS с публичным IP **или** Tailscale-аккаунт

---

## Быстрая установка

Открой Claude Code в любой папке (где удобно) и вставь одну фразу:

> Склонируй https://github.com/3702511-debug/ccdash.git в ~/cc-dashboard, прочитай DEPLOY.md в этой папке и помоги мне развернуть дашборд.

Claude склонирует репо, прочтёт инструкцию и проведёт по шагам. Спросит логин/пароль для дашборда, есть ли VPS для удалённого доступа с телефона, и т.д.

Никаких других файлов и команд от автора репозитория не нужно — всё внутри.

## Обновления

Когда автор репо запушит изменения:

```bash
cd cc-dashboard
./update.sh
```

Скрипт сам сделает `git pull`, переставит зависимости и перезапустит LaunchAgent.

## Если хочешь сам, без Claude

См. `DEPLOY.md` — там пошаговая инструкция со всеми командами.
